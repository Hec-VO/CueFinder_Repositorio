# CueFinder

## Introducción Profesional
Hola, soy un programador móvil y psicólogo biomédico dedicado a ayudar a las personas a mejorar su calidad de vida. Este repositorio contiene el código para la aplicación CueFinder, diseñada para apoyar a los usuarios en su camino hacia la recuperación.

## Fundamentos Científicos
Nuestra aplicación se basa en el modelo de Marlatt y el enfoque iRISA desarrollado por Goldstein-Volkow, lo que garantiza una base científica sólida en la prevención de recaídas.

## Público Objetivo
Esta aplicación está dirigida a mexicanos mayores de 18 años que buscan herramientas para el manejo de sus emociones y comportamientos.

## Enfoque Terapéutico
Utilizamos la terapia cognitiva conductual (TCC) y la prevención de recaídas basada en mindfulness como enfoques principales para ayudar a los usuarios.

## Cumplimiento Normativo
Nos adherimos a la Ley Federal de Protección de Datos, garantizando los derechos ARCO (Acceso, Rectificación, Cancelación y Oposición) y a los estándares de la APA (American Psychological Association) en sus secciones 3.10, 4.01 y 4.02.

## Proceso de Registro
Los usuarios deberán completar un sencillo proceso de registro y aceptar la interfaz de consentimiento informado para comenzar a utilizar la aplicación.

## Herramientas de Afrontamiento
La aplicación incluye 10 herramientas de afrontamiento, cada una con instrucciones específicas:
1. **Respiración 4-7-8**: Instrucciones sobre cómo realizar la técnica de respiración.
2. **Surfing del Urge**: Estrategia para enfrentar los deseos.
3. **Técnica 5-4-3-2-1**: Enfoque para centrar la atención en el presente.
4. **Postergación Consciente**: Manejo de impulsos.
5. **Lista de Razones**: Reflexiones para mantener la motivación.
6. **Reestructuración Cognitiva**: Cambio de patrones de pensamiento.
7. **Afirmaciones**: Declaraciones positivas.
8. **Caminata Consciente**: Ejercicio mediante atención plena.
9. **Visualización**: Técnicas para imaginar un futuro positivo.
10. **Sustitución Oral**: Alternativas saludables para satisfacer necesidades.

## Funciones para Psicólogos
Los psicólogos pueden verificar el código 012345, modificar herramientas, ajustar los tiempos de respiración y agregar o editar razones directamente desde la aplicación.

## Gestión de Datos
La aplicación incluye opciones para la eliminación irreversible de datos, asegurando la privacidad del usuario.

## Estructura del Repositorio
El repositorio está organizado de manera clara, facilitando la comprensión y el uso del código fuente.
