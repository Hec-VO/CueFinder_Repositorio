# Bibliografía

## 📖 Contenido

Esta carpeta contiene la documentación de referencias científicas y académicas utilizadas en el desarrollo de **CueFinder**.

### Documentos Incluidos

Los archivos PDF almacenados aquí incluyen:

- **Investigaciones científicas** sobre el tratamiento del tabaquismo
- **Estudios psicológicos** relacionados con el acompañamiento terapéutico
- **Protocolos y guías** de intervención en conductas adictivas
- **Literatura académica** de apoyo para profesionales de la salud mental

### 💡 Propósito

Estos documentos sirven como:
- Base teórica para la elaboración de la aplicación
- Referencias para profesionales que desean profundizar en los temas tratados
- Materiales de consulta para pacientes interesados en ampliar sus conocimientos
