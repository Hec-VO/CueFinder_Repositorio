# Prompt

## 🧠 Contenido

Esta carpeta contiene los **prompts utilizados para Google AI Studio** en el desarrollo e implementación de **CueFinder**.

### Descripción

Los prompts almacenados aquí incluyen:

- **Instrucciones base** para la generación de respuestas terapéuticas
- **Configuración de IA** utilizada en el procesamiento de interacciones
- **Parámetros de comportamiento** del modelo de inteligencia artificial
- **Ejemplos de conversaciones** para el entrenamiento y validación

### 🎯 Propósito

Estos prompts están documentados para:
- **Transparencia técnica** - Mostrar cómo funciona el motor de IA
- **Reproducibilidad** - Permitir que desarrolladores entiendan la lógica de la aplicación
- **Mejora continua** - Servir como base para optimizaciones futuras
- **Auditoría** - Facilitar la revisión de los procesos de generación de respuestas

### 📝 Nota

El refinamiento de estos prompts es un proceso continuo. Los cambios se registran para mantener un historial de evolución del modelo.
